## 📚 KENDİNİ GELİŞTİR

Bu projede çalışırken şunları öğrendin ya da öğreneceksin:

- **React** — modern web'in temel taşı
- **Tailwind CSS** — utility-first styling
- **Framer Motion** — animasyon kütüphanesi
- **i18next** — çoklu dil desteği
- **Git** — versiyon kontrolü

Bunları öğrendiysen sıradaki adımlar:

| Konu                                 | Kaynak                                                       |
| ------------------------------------ | ------------------------------------------------------------ |
| React ileri konular (hooks, context) | [react.dev](https://react.dev) — resmi dokümantasyon         |
| TypeScript                           | Mevcut projeyi TS'e geçirmek harika bir pratik               |
| Next.js                              | React'in production versiyonu, SEO için çok daha güçlü       |
| Figma                                | Tasarım araçları — Design Coord ile koordineli çalışmak için |
| Vercel / Netlify                     | Siteyi deploy etmek, CI/CD pipeline                          |
